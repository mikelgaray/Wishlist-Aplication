# Wish List Aplication

A simple wish list application where you can add, manage and keep track of the games you want.

## What it does

This app lets users create and manage their personal video game wish lists. You can add games you want to play, and once you get them, write a review with your thoughts and rating. Built as a backend Java application with persistent storage using MySQL.

## Tech Stack

- Java
- Hibernate (ORM)
- MySQL
